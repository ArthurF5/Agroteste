Agroteste
===========
Sistema web para gerenciamento de rotinas de um laboratório de análises de sementes.
[Agroteste](http:www.atager.com.br)

Tecnologias
-----------
- Lavavel com ACL
- PHP 7.1
- MySQL 5.5
- Git
- PhpMyAdmin 
- Composer
- cURL
- Vim
- Redis

Status
----------
Ainda em desenvolvimento.
